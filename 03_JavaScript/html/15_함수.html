함수(function)

    원 뜻은 기능이라는뜻
    - 목표하는 결과를 만들기 위한 코드의 묶음
    : 기능을 하게 하려고 쓰는 코드들

    - 코드를 {}(<-블록)으로 묶어 작성하고
         붙여진 이름을 호출해서 사용
    : html에서 준 이름 js 넘어가 서 {블록}에 넣어서 사용함

    * 함수 선언
     * 함수 정의
     * 함수 호출

     [기본 함수 형태]

     function 함수명(){  <-함수 선언(매개변수)
         // <-함수 정의
     }
     
     함수명(); // 함수 호출(전달인자)

매개 변수 (Parameter), 전달 인자(Arguments)

    * 매개 변수(Parameter)
     - 함수 선언 시 () 안에 작성되는 변수
      -> 전달 인자로 부터 받은 값을 저장
      -> 함수 정의 부분에서 사용 가능

    * 전달 인자(Arguments)
     - 함수 호출 시 () 안에 작성되는 값
      -> 함수 매개 변수로 전달되는 값
